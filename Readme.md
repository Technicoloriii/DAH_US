# 美国对外医疗援助研究 / U.S. Foreign Health Aid Study

本仓库旨在研究美国对外医疗援助的相关因素，包括援助资金分配、受援国特征以及援助效果分析。

This repository is dedicated to studying the factors related to U.S. foreign health aid, including the allocation of aid funds, the characteristics of recipient countries, and the analysis of aid effectiveness.

## 仓库内容 / Repository Content

本仓库包含以下内容：
- **数据 (Data)**：存放用于分析的原始数据和处理后的数据集。
- **代码 (Code)**：用于数据处理、分析和可视化的 Python 代码。
- **图表产出 (Charts and Visualizations)**：生成的地理地图和其他可视化图表，以便分析和展示援助模式。

This repository contains the following:
- **Data**: Raw datasets and processed datasets for analysis.
- **Code**: Python scripts used for data processing, analysis, and visualization.
- **Charts and Visualizations**: Generated maps and visualizations that help analyze and demonstrate aid distribution patterns.

## 研究背景 / Research Background

本研究旨在探讨美国对外医疗援助的决定因素，分析受援国的经济、地理和健康相关变量对援助的影响。通过数据分析和可视化，揭示美国在全球医疗援助中的分布模式和优先事项。

This research aims to explore the determining factors of U.S. foreign health aid by analyzing the economic, geographic, and health-related variables of recipient countries. Through data analysis and visualization, we aim to uncover patterns and priorities in U.S. global health aid distribution.

## 使用方法 / Usage

- 你可以通过仓库中的代码文件运行相关的分析，并生成对应的地图和图表。
- 数据文件存放在 `data` 文件夹中，包含用于分析的数据集。
- 可视化结果存放在 `output` 文件夹中，包括地图和图表。

- You can run the relevant analyses by using the code files in the repository and generate corresponding maps and charts.
- Data files are stored in the `data` folder, containing the datasets used for analysis.
- Visualization results are stored in the `output` folder, including maps and charts.
